# My-Grove-

A local-first personal growth and project management system built with Obsidian Markdown and AI-assisted workflows.

## v0.1

My Grove starts as a simple Obsidian-first Markdown space for collecting, growing, and connecting ideas.

This version is intentionally small:

- Obsidian Markdown files only
- No app
- No database
- No login or auth
- No complex automation
