# Agent Plan

Three lightweight agents shaped the v0.1 structure.

## Agent 1: Grove Architect

Task: create a clear folder structure that feels natural in Obsidian.

Decision: use a simple maturity path from inbox to evergreen notes, plus projects and archive.

## Agent 2: Obsidian Gardener

Task: make the Markdown easy to navigate without adding tooling.

Decision: use plain Markdown, Obsidian wikilinks, README index notes, and simple templates.

## Agent 3: v0.1 Guardian

Task: keep the release small and inside the stated constraints.

Decision: avoid app code, database files, auth, automation, package manifests, and Obsidian plugin config.

## Result

The first version is a Markdown-only vault skeleton that can grow later without forcing architecture too early.
